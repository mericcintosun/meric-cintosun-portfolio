# Portfolio Website

A modern and interactive portfolio website built with Next.js and React, featuring a beautiful UI, 3D elements, and various interactive components.

## 🚀 Technologies Used

- **Frontend Framework:** Next.js 15.0
- **UI Library:** React 18
- **Styling:** 
  - Tailwind CSS
  - Material Tailwind
- **3D Graphics:**
  - Three.js
  - React Three Fiber
  - React Three Drei
- **Animations & Effects:**
  - Motion
  - TSParticles
  - React Vertical Timeline
- **Email & Contact:**
  - Nodemailer
  - Resend
  - React Phone Input 2
- **Icons:**
  - Hero Icons
  - React Icons

## ✨ Features

- Modern and responsive design
- Interactive 3D elements and animations
- Particle effects for enhanced visual appeal
- Blog section with dynamic content
- Project showcase with detailed cards
- Skills and experience timeline
- Contact form with email functionality
- About section with personal information
- Smooth page transitions and animations

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd portfolio-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Environment Setup**
   - Create a `.env.local` file in the root directory
   - Add necessary environment variables:
     ```env
     # Add your environment variables here
     # Example:
     # RESEND_API_KEY=your_resend_api_key
     ```

4. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. **Build for production**
   ```bash
   npm run build
   # or
   yarn build
   ```
